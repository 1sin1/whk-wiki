# 使用技巧

import Geogebra from '@site/src/GeoGebra';

本站采用 $\KaTeX$ 渲染公式．本站所有公式均可通过直接选中复制源码．可以测试一下复制效果：

$$
\begin{cases}
f(x) = \sin x + \cos x \\
\pu{57.3kJ*mol-1} \\
\ce{2H2O ->[点燃] 2H2 ^ + O2 ^} \\
\end{cases}
$$

支持白天 / 夜间模式．综合阅读体验和加粗提示效果的考量，更推荐白天模式．

部分页面内嵌 Geogebra 文件，可以与它动态交互：
