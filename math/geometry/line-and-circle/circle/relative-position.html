圆相关位置关系

点与圆的位置关系​

点与圆的位置关系内容​

点 PPP 和 ⊙A\odot A⊙A 的位置关系有三种：PPP 在 ⊙A\odot A⊙A 外；PPP 在 ⊙A\odot A⊙A 内；PPP 在 ⊙A\odot A⊙A 上．

设 PPP 到圆心 AAA 的距离为 ddd，⊙A\odot A⊙A 的半径为 rrr，则：

  • PPP 在 ⊙A\odot A⊙A 外，当且仅当 d>rd > rd>r．
  • PPP 在 ⊙A\odot A⊙A 内，当且仅当 d<rd < rd<r．
  • PPP 在 ⊙A\odot A⊙A 上，当且仅当 d=rd = rd=r．

标准式上快速判断点与圆位置关系​

⊙A\odot A⊙A：(x−xA)2+(y−yA)2=r2(x - x_A)^2 + (y - y_A)^2 = r^2(x−xA​)2+(y−yA​)2=r2．快速判断 P(x0,y0)P(x_0, y_0)P(x0​,y0​) 和 ⊙A\odot A⊙A 的大小关系．

我们只需将 (x0,y0)(x_0, y_0)(x0​,y0​) 代入上面方程的等号左边部分，就可以得到 PPP 到圆心 AAA 的距离的平方 d2d^2d2．然后，我们将它和方程等号右边的 r2r^2r2 进行比较即可：

  • d2>r2d^2 > r^2d2>r2，当且仅当 d>rd > rd>r，当且仅当 PPP 在 ⊙A\odot A⊙A 外．
  • d2<r2d^2 < r^2d2<r2，当且仅当 d<rd < rd<r，当且仅当 PPP 在 ⊙A\odot A⊙A 内．
  • d2=r2d^2 = r^2d2=r2，当且仅当 d=rd = rd=r，当且仅当 PPP 在 ⊙A\odot A⊙A 上．
例题 1.2

判断点 (1,−3)(1, -3)(1,−3) 与圆 (x−2)2+(y+1)2=6(x - 2)^2 + (y + 1)^2 = 6(x−2)2+(y+1)2=6 的位置关系．

例题 1.2 解答

(1−2)2+(−3+1)2=5<6(1 - 2)^2 + (-3 + 1)^2 = 5 < 6(1−2)2+(−3+1)2=5<6．因此点在圆内．

一般式上快速判断点与圆位置关系​

⊙A\odot A⊙A：x2+y2+Dx+Ey+F=0x^2 + y^2 + Dx + Ey + F = 0x2+y2+Dx+Ey+F=0．快速判断 P(x0,y0)P(x_0, y_0)P(x0​,y0​) 和 ⊙A\odot A⊙A 的大小关系．

根据圆的一般方程的知识，上面的方程可以变形为 \left(x + \dfrac D 2\right)^2 + \left(y + \dfrac E 2\right)^2 - \dfrac{D^2 + E^2 - 4F}{4} = 0，所以一般式方程的本质就是 d2−r2=0d^2 - r^2 = 0d2−r2=0，方程左边的部分就是 d2−r2d^2 - r^2d2−r2．

因此我们可以仍考虑将点的坐标直接代入方程左半部分，算出 d2−r2d^2 - r^2d2−r2 的值，然后和 000 比较．>0> 0>0 说明在圆外；=0= 0=0 说明在圆上；<0< 0<0 说明在圆内．

判断点 (1,1)(1, 1)(1,1) 与圆 x2+y2−4x−8y+15=0x^2 + y^2 - 4x - 8y + 15 = 0x2+y2−4x−8y+15=0 的位置关系．

例题 1.3 解答

平方项系数必须统一为 111

对于平方项系数没统一为 111 的情形，即 Ax2+Ay2+Dx+Ey+F=0Ax^2 + Ay^2 + Dx + Ey + F = 0Ax2+Ay2+Dx+Ey+F=0，此时方程等号左边表示的是 A(d2−r2)A(d^2 - r^2)A(d2−r2)．我们如果直接将 P(x0,y0)P(x_0, y_0)P(x0​,y0​) 坐标代入等号左边，得到的值是 A(d2−r2)A(d^2 - r^2)A(d2−r2) 的正负．A<0A < 0A<0，其正负性和 d2−r2d^2 - r^2d2−r2 相反．

这也就意味着在 A<0A < 0A<0 的情形下，我们算出方程左边的值后，和 000 比较大小时不能继承先前的结论（得到的是反的结果）．正确的做法是将方程统一成 A>0A > 0A>0（建议直接统一成一般式形式 A=1A = 1A=1）再判断．

直线与圆的位置关系​

直线与圆的位置关系内容​

⊙A\odot A⊙A 和直线 lll 的位置关系有三种：lll 与 ⊙A\odot A⊙A 相离；lll 与 ⊙A\odot A⊙A 相交；lll 与 ⊙A\odot A⊙A 相切．

设圆心 AAA 到 lll 的距离为 ddd，⊙A\odot A⊙A 的半径为 rrr，则：

  • lll 与 ⊙A\odot A⊙A 相离，当且仅当 d>rd > rd>r，当且仅当 lll 与 ⊙A\odot A⊙A 没有交点．
  • lll 与 ⊙A\odot A⊙A 相交，当且仅当 d<rd < rd<r，当且仅当 lll 与 ⊙A\odot A⊙A 有两个交点．
  • lll 与 ⊙A\odot A⊙A 相切，当且仅当 d=rd = rd=r，当且仅当 lll 与 ⊙A\odot A⊙A 恰有一个交点．

判断直线与圆的位置关系有两种角度，对应地，在解析几何中我们也有两套判断方式．

距离法​

我们可以从距离的角度判断直线与圆的位置关系．用点到直线的距离公式算出 ddd，直接和 rrr 比较即可．

判别式法​

还可以从交点数量的角度判断直线与圆的位置关系．我们考虑联立两个方程组：

{Ax+By+C=0x2+y2+Dx+Ey+F=0\begin{cases} Ax + By + C = 0\\ x^2 + y^2 + Dx + Ey + F = 0 \end{cases}{Ax+By+C=0x2+y2+Dx+Ey+F=0​

这虽然是一个二元二次方程组，但有一个方程是一次的．令 B≠0B \ne 0B=0，此时 yyy 可以由 xxx 线性表出；然后我们将第二个方程的 yyy 全都代换为 xxx，于是第二个方程变成一元二次方程．

形式化地，当 B≠0B \ne 0B=0 时，我们可以将方程等价变形为：

{y=f1(x)f2(x,y)=0\begin{cases} y = f_1(x) \\ f_2(x, y) = 0 \end{cases}{y=f1​(x)f2​(x,y)=0​

然后等价变形为：

{y=f1(x)f2(x,f(x))=0\begin{cases} y = f_1(x) \\ f_2(x, f(x)) = 0 \end{cases}{y=f1​(x)f2​(x,f(x))=0​

根据第一个方程，一个 xxx 恰好对应一个 yyy．那么，f2(x,f(x))=0f_2(x, f(x)) = 0f2​(x,f(x))=0 方程解的个数就是整个方程解的个数；整个方程解的个数就是圆与直线的交点总数．因此，我们只需判断 f2(x,f(x))=0f_2(x, f(x)) = 0f2​(x,f(x))=0 方程解的个数．

f2(x,f(x))=0f_2(x, f(x)) = 0f2​(x,f(x))=0 是一个二元一次方程，判断其解的个数，我们可以判断 Δ\DeltaΔ 的正负性．

  • Δ=0\Delta = 0Δ=0 时有一个解，对应一个交点，对应相切．
  • Δ>0\Delta > 0Δ>0 时有两个解，对应两个交点，对应相交．
  • Δ<0\Delta < 0Δ<0 时有零个解，对应零个交点，对应相离．
例题 3.3

判断直线 x−y+4=0x - y + 4 = 0x−y+4=0 与圆 x2+y2=8x^2 + y^2 = 8x2+y2=8 的位置关系．

例题 3.3 解答

联立方程组：

{x−y+4=0x2+y2=8\begin{cases} x - y + 4 = 0 \\ x^2 + y^2 = 8 \end{cases}{x−y+4=0x2+y2=8​

变形为

{y=x+4x2+(x+4)2=8\begin{cases} y = x + 4 \\ x^2 + (x + 4)^2 = 8 \end{cases}{y=x+4x2+(x+4)2=8​

该方程解的个数与 x2+(x+4)2=8x^2 + (x + 4)^2 = 8x2+(x+4)2=8 中解的个数相同．整理得 x2+4x+4=0x^2 + 4x + 4 = 0x2+4x+4=0，计算其判别式 Δ=42−4×4=0\Delta = 4^2 - 4 \times 4 = 0Δ=42−4×4=0，因此该方程只有一个解，原方程组只有一个解，直线与圆相切．

该方法可将直线与圆的交点坐标顺便解出．但这里有一个注意事项．来看下面的错误示范：

例题 3.3

判断直线 x−y+4=0x - y + 4 = 0x−y+4=0 与圆 x2+y2=8x^2 + y^2 = 8x2+y2=8 的位置关系．

例题 3.3 错误解答示范

联立方程组：

{x−y+4=0x2+y2=8\begin{cases} x - y + 4 = 0 \\ x^2 + y^2 = 8 \end{cases}{x−y+4=0x2+y2=8​

变形为

{y=x+4x2+(x+4)2=8\begin{cases} y = x + 4 \\ x^2 + (x + 4)^2 = 8 \end{cases}{y=x+4x2+(x+4)2=8​

代入消元，第二个方程变形为 x2+4x+4=0x^2 + 4x + 4 = 0x2+4x+4=0，解得 x=−2x = -2x=−2．将 x=−2x = -2x=−2 代入第二个方程后，有 (−2)2+y2=8(-2)^2 + y^2 = 8(−2)2+y2=8，解得 y=2y = 2y=2 或 y=−2y = -2y=−2．

因此，直线与圆有两个交点，直线与圆相交．

上面做法的错误之处在于：

{y=x+4x2+(x+4)2=8\begin{cases} y = x + 4 \\ x^2 + (x + 4)^2 = 8 \end{cases}{y=x+4x2+(x+4)2=8​

其等价的结果应为：

{y=x+4x=2\begin{cases} y = x + 4 \\ x = 2 \end{cases}{y=x+4x=2​

很明显这里 x=2x = 2x=2 的结果应该代入第一个方程算出 yyy，而不应该代入原先的第二个方程．

刚刚的错误示例提醒我们：方程组并不总是利用一个方程求出一个变量后，将这个解随便代入一个原方程求出另一个变量；在绝大多数多元一次方程组我们可以这么做，但二次方程组就不一样了．

在刚刚的问题中，利用圆方程解出 xxx 后，必须重新代入 直线方程 才能得到正确的 yyy．

关于这种问题的更多细节，如「为什么绝大多数多元一次方程组，解出一个元后随便代入一个方程就能解出另一元，但二次方程组就不行了？」这种问题，可以在方程页面得到解答．

另外，最开始我们限制 B≠0B \ne 0B=0，那 B=0B = 0B=0 怎么办呢？此时一定有 A≠0A \ne 0A=0，反过来将 xxx 用 yyy 线性表出即可．

直线与圆的位置关系 两种方法比较​

笔者的推荐是：首选距离法．判别式法与距离法相比计算量一般是更大的，尤其当方程含参时．

当且仅当询问直线和圆的交点坐标时，我们再联立两个方程求交点坐标．

鉴于带着参数操作方程计算量很大，询问含参直线与圆的切点坐标时，通常先用距离法把参数求出来，再联立方程求坐标．

事实上，单纯运用判别式判定交点数几乎不用，因为使用判别式法的情景通常都是要进一步求具体交点坐标．

圆与圆的位置关系​

圆与圆的位置关系内容​

两圆之间的位置关系共有五种：外离；外切；相交；内切；内含．设两圆半径分别为 r1r_1r1​，r2r_2r2​，两圆圆心的距离（圆心距）为 ddd，则：

  • d>r1+r2d > r_1 + r_2d>r1​+r2​ 时，两圆外离．
  • d=r1+r2d = r_1 + r_2d=r1​+r2​ 时，两圆外切．
  • ∣r1−r2∣<d<r1+r2|r_1 - r_2| < d < r_1 + r_2∣r1​−r2​∣<d<r1​+r2​ 时，两圆相交．
  • d=∣r1−r2∣d = |r_1 - r_2|d=∣r1​−r2​∣ 时，两圆内切．
  • d<∣r1−r2∣d < |r_1 - r_2|d<∣r1​−r2​∣ 时，两圆内含．

特别地：

  • 当 d=0d = 0d=0 时，两圆互为同心圆（它们的圆心相同）．
  • 当 r1=r2r_1 = r_2r1​=r2​ 时，两圆没有内含的情形，内切等价于重合．

因此，确定两圆的位置关系，我们要比较弦心距与半径和、半径差两个量的大小关系．

圆与圆的位置关系题型​

两圆上两动点距离值域​

已知两个圆 ⊙C1\odot C_1⊙C1​ 和 ⊙C2\odot C_2⊙C2​，⊙C1\odot C_1⊙C1​ 上有一动点 P1P_1P1​，⊙C2\odot C_2⊙C2​ 上有一动点 P2P_2P2​，求 P1P_1P1​ 和 P2P_2P2​ 间距离的取值范围．
