基本求通项问题

先猜后证（数学归纳法）​

例题 1

已知数列 aaa 满足 a1=1a_1 = 1a1​=1，an=nan+nana_n = \sqrt{na_n} + \dfrac n {a_n}an​=nan​​+an​n​，求 ana_nan​．

乍一看什么思路都没有，不妨算几个数．a1=1a_1 = 1a1​=1，a2=2a_2 = 2a2​=2，a3=3a_3 = 3a3​=3……因此可以猜测 an=na_n = nan​=n．

证明一个通项公式成立，一般采用数学归纳法．

例题 1 解答

证明 an=na_n = nan​=n 对任意正整数 nnn 都成立：

n=1n = 1n=1 时 a1=1a_1 = 1a1​=1，成立．

对任意 k∈N∗k \in \N^\astk∈N∗，若 ak=ka_k = kak​=k，则 ak+1=kak+kak=k+1a_{k + 1} = \sqrt{ka_k} + \dfrac k {a_k} = k + 1ak+1​=kak​​+ak​k​=k+1．成立．

因此对所有 nnn 均有 an=na_n = nan​=n，an=na_n = nan​=n 为 aaa 的通项公式，证毕．

算术结构相同​

如果题目给出的式子中，数列每一项在式子中出现的结构相同，可以考虑将这个结构本身 设成新数列．

思想重要性提示

不局限于求通项，做各种数列问题时，这个思想都是 非常常用，非常重要的，请读者 务必对算术结构相同敏感．

例题 2.1

已知 a1=2a_1 = 2a1​=2，2an−2an−1=1\sqrt{2a_n} - \sqrt{2a_{n - 1}} = 12an​​−2an−1​​=1，求 ana_nan​．

例题 2.1 解答

设 bn=2anb_n = \sqrt{2a_n}bn​=2an​​，则 bn−bn−1=1b_n - b_{n - 1} = 1bn​−bn−1​=1，bbb 为公差为 111 的等差数列．

a1=2a_1 = 2a1​=2 可知 b1=2b_1 = 2b1​=2，因此 bn=n+1b_n = n + 1bn​=n+1．

an=bn22a_n = \dfrac{{b_n}^2} 2an​=2bn​2​，即 an=n2+2n+12a_n = \dfrac{n^2 + 2n + 1}2an​=2n2+2n+1​．

例题 2.2

已知数列 aaa 满足 a1+3a2+5a3+⋯+(2n−1)an=4na_1 + 3a_2 + 5a_3 + \cdots + (2n - 1)a_n = 4na1​+3a2​+5a3​+⋯+(2n−1)an​=4n，求 ana_nan​．

例题 2.2 解答

设 bn=(2n−1)anb_n = (2n - 1)a_nbn​=(2n−1)an​，则题目相当于给出 bbb 的前缀和 Sn=4nS_n = 4nSn​=4n．

差分可得 bn=4b_n = 4bn​=4．因此 an=42n−1a_n = \dfrac 4 {2n - 1}an​=2n−14​．

例题 2.3

已知正项数列 aaa 满足 2an2=an−12+an+122{a_n}^2 = {a_{n - 1}}^2 + {a_{n + 1}}^22an​2=an−1​2+an+1​2 对任意 n∈N∗,n≥2n \in \N^\ast, n \ge 2n∈N∗,n≥2 成立，a1=1a_1 = 1a1​=1，a2=2a_2 = 2a2​=2，求 a100a_{100}a100​ 的值．

例题 2.3 解答

令 bn=an2b_n = {a_n}^2bn​=an​2，则有 2bn=bn−1+bn+12b_n = b_{n - 1} + b_{n + 1}2bn​=bn−1​+bn+1​，因此 bbb 是等差数列．

而 b1=a12=1b_1 = {a_1}^2 = 1b1​=a1​2=1，b2=a22=4b_2 = {a_2}^2 = 4b2​=a2​2=4，可知 bn=3n−2b_n = 3n - 2bn​=3n−2．

由于 aaa 为正项数列，因此 a100=b100=98=72a_{100} = \sqrt{b_{100}} = \sqrt{98} = 7 \sqrt 2a100​=b100​​=98​=72​．

累加法 / 累乘法​

如果题目给出的 aaa 的递推公式 形如 或 可以变形为 这两种形式：

  • an−an−1=f(n)a_n - a_{n - 1} = f(n)an​−an−1​=f(n)．
  • anan−1=f(n)\dfrac{a_n}{a_{n - 1}} = f(n)an−1​an​​=f(n)．

则前者相当于 给出了 aaa 的差分 DDD 的通项公式，后者相当于 给出了 aaa 的「比分」RRR 的通项公式．差分做前缀和，「比分」做前缀积，即可得到 aaa 的通项公式．

例题 3.1

已知数列 aaa 满足 a1=2a_1 = 2a1​=2 且 an+1=an+n+1a_{n + 1} = a_n + n + 1an+1​=an​+n+1，求 ana_nan​．

例题 3.1 解答

条件变形得 an+1−an=n+1a_{n + 1} - a_n = n + 1an+1​−an​=n+1．

当 n≥2n \ge 2n≥2 时，

an=a1+(a2−a1)+(a3−a2)+⋯+(an−an−1)=2+2+3+⋯+n=2+n2+n−22=n2+n+22\begin{aligned} a_n &= a_1 + (a_2 - a_1) + (a_3 - a_2) + \cdots + (a_n - a_{n - 1}) \\ &= 2 + 2 + 3 + \cdots + n \\ & = 2 + \dfrac{n^2 + n - 2} 2 \\ & = \dfrac{n^2 + n + 2} 2 \end{aligned}an​​=a1​+(a2​−a1​)+(a3​−a2​)+⋯+(an​−an−1​)=2+2+3+⋯+n=2+2n2+n−2​=2n2+n+2​​

当 n=1n = 1n=1 时，a1=2a_1 = 2a1​=2 符合上式．

因此 an=n2+n+22a_n = \dfrac{n^2 + n + 2}2an​=2n2+n+2​．

还记得这里为什么要分类讨论吗？其实是在基础讲过的内容，这里回顾一下：

这里分开算一下的目的是，n=1n = 1n=1 时后面那个 2+⋯+n2 + \cdots + n2+⋯+n 的 数列为空，数列为空的情形习惯上分类讨论一下，因为各种求和公式理论上都是在数列存在时推导的．

然而事实上，等差数列和等比数列的求和公式在代入项数 =0= 0=0 时计算出来的和均为 000，是符合期望的，因此两种情况往往可以合并．

  • 刚才又讲了个 裂项求和，那么题目就可以把数列的差分设计为可以裂项求和的分式．裂项和代入项数 =0= 0=0 结果 可能不为 000，结果可能不能合并，是需要注意的．
  • 而 等差求和 与 等比求和 最终 基本是可以合并的，然而在写过程时也需注意分开写，这是应试的要求．
例题 3.2

已知数列 aaa 满足 a1=1a_1 = 1a1​=1，(n+2)an=nan−1(n + 2)a_n = na_{n - 1}(n+2)an​=nan−1​，求 ana_nan​．

例题 3.2 解答

容易看出 aaa 不包含零，因此条件变形

anan−1=nn+2\dfrac{a_n}{a_{n - 1}} = \dfrac{n}{n + 2}an−1​an​​=n+2n​

当 n≥2n \ge 2n≥2 时，

an=a1⋅(a2a1)⋅(a3a2)⋯(anan−1)=24×35×⋯×nn+2=2×3×⋯×n4×5×⋯×(n+2)\begin{aligned} a_n &= a_1 \cdot \left(\dfrac{a_2}{a_1}\right) \cdot \left(\dfrac{a_3}{a_2}\right) \cdots \left(\dfrac{a_n}{a_{n - 1}}\right) \\ &= \dfrac 2 4 \times \dfrac 3 5 \times \cdots \times \dfrac{n}{n + 2} \\ &= \dfrac{2 \times 3 \times \cdots \times n}{4 \times 5 \times \cdots \times (n + 2)} \end{aligned}an​​=a1​⋅(a1​a2​​)⋅(a2​a3​​)⋯(an−1​an​​)=42​×53​×⋯×n+2n​=4×5×⋯×(n+2)2×3×⋯×n​​

这里可以直接约分理解，但是也有类似「裂项求和相消」的逻辑疏漏，存在 n=2n = 2n=2 的边界情况．

此时可以考虑「滑动窗口商」：对任意非零数列 bbb，设 [l,r][l, r][l,r] 为 ∏i=lrbi\prod_{i = l}^r b_i∏i=lr​bi​，则

[1,n][k+1,n+k]=[1,k][n+1,n+k]\dfrac{[1, n]}{[k + 1, n + k]} = \dfrac{[1, k]}{[n + 1, n + k]}[k+1,n+k][1,n]​=[n+1,n+k][1,k]​

2×3×⋯×n4×5×⋯×(n+2)\dfrac{2 \times 3 \times \cdots \times n}{4 \times 5 \times \cdots \times (n + 2)}4×5×⋯×(n+2)2×3×⋯×n​ 是一个窗口长度为 n−1n - 1n−1，偏移距离为 222 的「滑动窗口商」，其等于窗口长度为 222，偏移距离为 n−1n - 1n−1 的「滑动窗口商」，即 2×3(n+1)(n+2)=6n2+3n+2\dfrac{2 \times 3}{(n + 1)(n + 2)} = \dfrac 6 {n^2 + 3n + 2}(n+1)(n+2)2×3​=n2+3n+26​．

n=1n = 1n=1 时 a1=1a_1 = 1a1​=1，符合上式．

因此 an=6n2+3n+2a_n = \dfrac 6{n^2 + 3n + 2}an​=n2+3n+26​．

最后再次注意 算术结构相同 思想的广泛性：如这种式子

1an−1an−1=n+1\dfrac 1 {a_n} - \dfrac 1 {a_{n - 1}} = n + 1an​1​−an−1​1​=n+1

应立刻注意到设 bn=1anb_n = \dfrac 1 {a_n}bn​=an​1​，则 bnb_nbn​ 差分可知，通项可求，进而 ana_nan​ 通项也可求．

有提示构造​

例题 4.1

数列 aaa 满足 a1=−12a_1 = -\dfrac 1 2a1​=−21​，2an=an−1−n−12a_n = a_{n - 1} - n - 12an​=an−1​−n−1．

  • （1）求证：{an+n}\{a_n + n\}{an​+n} 是等比数列．
  • （2）求 ana_nan​．

如果没有第一问，直接求 aaa 的通项，在不了解方法之前是比较困难的．而本题出题人通过设第一小问的方式，提示我们 {an+n}\{a_n + n\}{an​+n} 是等比数列．那么很明显．设 bn=an+nb_n = a_n + nbn​=an​+n，只要我们知道 bbb 的首项与公比，则 bbb 的通项可解；而 bbb 的通项减去 nnn 就是 aaa 的通项了．

对于未知通项的 aaa，通过 构造并证明 aaa 衍生（换元）出的另一数列 bbb 为 可解通项数列 的方式，从而获取 bbb 的通项，反推 aaa 的通项的方式称作 构造法．而上述题目通过设小问的方式，直接提示了构造方向．因此只需按部就班操作．

提示：读者需要对证明等比数列的操作异常熟悉，对此不熟悉的请见判定等比数列．

例题 4.1 解答

（1）令 bn=an+nb_n = a_n + nbn​=an​+n，将条件向目标变形：

2an=an−1−n−1  ⟺  2an+2n=an−1+n−1  ⟺  2bn=bn−1  ⟺  bn=12bn−12a_n = a_{n - 1} - n - 1 \iff 2a_n + 2n = a_{n - 1} + n - 1 \iff 2b_n = b_{n - 1} \iff b_n = \dfrac 1 2 b_{n - 1}2an​=an−1​−n−1⟺2an​+2n=an−1​+n−1⟺2bn​=bn−1​⟺bn​=21​bn−1​

验证可知 b1=−12+1=12≠0b_1 = -\dfrac 1 2 + 1 = \dfrac 1 2 \ne 0b1​=−21​+1=21​=0，公比 12≠0\dfrac 1 2 \ne 021​=0，因此 bbb 为首项 12\dfrac 1 221​，公比 12\dfrac 1 221​ 的等比数列．

（2）由（1）可知 bn=12nb_n = \dfrac 1 {2^n}bn​=2n1​．

bn=an+nb_n = a_n + nbn​=an​+n，可知 an=bn−n=12n−na_n = b_n - n = \dfrac 1 {2^n} - nan​=bn​−n=2n1​−n．

例题 4.2

已知数列 aaa 满足 a1=4a_1 = 4a1​=4，nan+1−(n+1)an=2n2+2nna_{n + 1} - (n + 1)a_n = 2n^2 + 2nnan+1​−(n+1)an​=2n2+2n．

  • （1）求证：数列 {ann}\left\{\dfrac{a_n} n\right\}{nan​​} 是等差数列．
  • （2）求 ana_nan​．
例题 4.2 解答

（1）令 bn=annb_n = \dfrac{a_n}nbn​=nan​​，对条件同时除以 n(n+1)n(n + 1)n(n+1) 可得

an+1n+1−ann=2\dfrac{a_{n + 1}}{n + 1} - \dfrac{a_n}n = 2n+1an+1​​−nan​​=2

即

bn+1−bn=2b_{n + 1} - b_n = 2bn+1​−bn​=2

可知 bbb 为公差为 222 的等差数列，证毕．

（2）b1=a11=4b_1 = \dfrac{a_1}1 = 4b1​=1a1​​=4，因此 bn=2n+2b_n = 2n + 2bn​=2n+2．

bn=annb_n = \dfrac{a_n}nbn​=nan​​，可知 an=nbn=n(2n+2)=2n2+2na_n = nb_n = n(2n + 2) = 2n^2 + 2nan​=nbn​=n(2n+2)=2n2+2n．

没有第一问这种提示的构造会在构造法求通项给出方法．

奇偶项数列​

首先介绍两个套路．

  • 数列 aaa 中，如果对于任意 n∈N∗n \in \N^\astn∈N∗ 有 an+an+1=f(n)a_n + a_{n + 1} = f(n)an​+an+1​=f(n)，则 an+2−an=f(n+1)−f(n)a_{n + 2} - a_n = f(n + 1) - f(n)an+2​−an​=f(n+1)−f(n)．
  • 非零数列 aaa 中，如果对于任意 n∈N∗n \in \N^\astn∈N∗ 有 anan+1=f(n)a_na_{n + 1} = f(n)an​an+1​=f(n)，则 an+2an=f(n+1)f(n)\dfrac{a_{n + 2}}{a_n} = \dfrac{f(n + 1)}{f(n)}an​an+2​​=f(n)f(n+1)​．

以第一条为例说明：将 an+1+an+2=f(n+1)a_{n + 1} + a_{n + 2} = f(n + 1)an+1​+an+2​=f(n+1) 与 an+an+1=f(n)a_n + a_{n + 1} = f(n)an​+an+1​=f(n) 作差即得．

这两个套路说明：邻项和 可以转化为 隔项差，邻项积 可以转化为 隔项商．
