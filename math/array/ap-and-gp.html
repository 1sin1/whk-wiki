等差数列基础​

等差数列基本概念​

如果一个数列从第二项起，每一项与前一项的差都为一个常数 ddd（difference），则将这个数列称作 等差数列，ddd 称作该数列的 公差．

即对于等差数列 aaa，有

D(a)n={a1,n=1d,n≥2D(a)_n = \begin{cases} a_1, & n = 1 \\ d, & n \ge 2 \end{cases}D(a)n​={a1​,d,​n=1n≥2​

对 D(a)D(a)D(a) 做前缀和可得

an=a1+(n−1)da_n = a_1 + (n - 1)dan​=a1​+(n−1)d

很明显，确定 a1a_1a1​ 和 ddd 后，等差数列确定，而只确定其中之一则无法确定等差数列．

对于长度为 333 的等差数列 a,b,ca, b, ca,b,c，称 bbb 是 aaa 和 ccc 的 等差中项．等差中项等价于算术平均值，即

b=a+c2b = \dfrac{a + c}2b=2a+c​

或

2b=a+c2b = a + c2b=a+c

可以由 b−a=c−bb - a = c - bb−a=c−b 简单变形推出．

等差数列的英文是 Arithmetic Progression，简称 AP．Arithmetic 意为「算术的」，正与这里的「算术平均值」对应．

等比数列基础​

等比数列基本概念​

如果一个数列从第二项起，每一项与前一项的比都为一个常数 qqq（quotient），则将这个数列称作 等比数列，qqq 称作该数列的 公比．

根据定义可以得知：

  • 一个等比数列中，每一项都不为 000．
  • q≠0q \ne 0q=0．

对于等比数列 aaa，有

R(a)n={a1,n=1q,n≥2R(a)_n = \begin{cases} a_1, & n = 1 \\ q, & n \ge 2 \end{cases}R(a)n​={a1​,q,​n=1n≥2​

对 R(a)R(a)R(a) 做前缀积可得

an=a1qn−1a_n = a_1 q^{n - 1}an​=a1​qn−1

确定 a1a_1a1​ 和 qqq 后，等比数列确定，而只确定其中之一则无法确定等比数列．

对于长度为 333 的等比数列 a,b,ca, b, ca,b,c，称 bbb 是 aaa 和 ccc 的 等比中项．这等价于

ba=cb\dfrac b a = \dfrac c bab​=bc​

又等价于

{b2=acabc≠0\begin{cases} b^2 = ac \\ abc \ne 0 \end{cases}{b2=acabc=0​

单向推出而不等价于 b2=acb^2 = acb2=ac．

也就是说，bbb 是 aaa 和 ccc 的等比中项可以推出 b2=acb^2 = acb2=ac，但反过来，知道 b2=acb^2 = acb2=ac 还不足以推出 bbb 是 aaa 和 ccc 的等比中项，必须再加条件 abc≠0abc \ne 0abc=0．这是与等差中项不同的地方．

还有一个明显不同的地方是：对于任意 aaa 和 ccc，其一定恰存在一个等差中项 bbb；但对于等比数列：

  • 如果 ac>0ac > 0ac>0，aaa 与 ccc 存在两个等比中项 b=±acb = \pm \sqrt{ac}b=±ac​．
  • 如果 ac<0ac < 0ac<0，aaa 与 ccc 不存在等比中项．

如 111 和 999 的等比中项为 ±3\pm 3±3，333 和 −3-3−3 都可以作为 111 和 999 的等比中项；而 111 和 −9-9−9 不存在等比中项．

等比数列的英文是 Geometric Progression，简称 GP．Geometric 意为「几何的」，正与这里 b=±acb = \pm\sqrt{ac}b=±ac​ 的「几何平均值」（采正值）对应．

等比数列的性质​

下设 aaa 为等比数列．

等比数列性质零​

这是一个排除解的小技巧，基于等比数列较为基本的性质，因此列在这里．

  • q>0q > 0q>0 时，数列 恒负 或 恒正．
  • q<0q < 0q<0 时，数列 正负邻项交替．
  • 无论如何，数列中 下标差为偶数 的两项 正负性一定相同．

利用这个性质，我们可以及时排除正负性错误的解．比如已知 a3=5a_3 = 5a3​=5，则 a5a_5a5​ 不可能是负数．如果解出 a5=2a_5 = 2a5​=2 或 a5=−1a_5 = -1a5​=−1，就可以直接排除后者了．

anam=qn−m\dfrac{a_n}{a_m} = q^{n - m}am​an​​=qn−m

an=amqn−ma_n = a_m q^{n - m}an​=am​qn−m

定义「等比乘除式」两个属性：

  • 次数和：即所有 aaa 的 次数 之和．
  • 下标和：即所有 aaa 的 下标与次数的积 之和．

如 a92a10\dfrac{{a_9}^2}{a_{10}}a10​a9​2​ 的次数和为 2−1=12 - 1 = 12−1=1，下标和为 9×2+10×(−1)=89 \times 2 + 10 \times (-1) = 89×2+10×(−1)=8．

结论：对于两个「等比乘除式」，如果它们的次数和与下标和均分别相同，则结果一定相同．这就是等比数列的 下标守恒定律．

证明：

对于等比数列 aaa，有

  • 当 q=−1q = -1q=−1 时，有且仅有正偶数 nnn 为解．
  • 否则，方程无解．

等比数列题型​

等比数列信息求解​

基本策略：通过将未知量转为 a1a_1a1​ 和 qqq 的式子，将条件转为若干 a1a_1a1​ 和 qqq 构成的二元方程组，然后解方程组．

与等差数列不同的是，等比数列构成的方程次数一般较高，需要注意良好的降次策略以及多解问题．

简化性质也要适度使用．

等比数列的判断与证明​

等比数列的证明有两种有细微差别的策略：

  • 证明 an+1an=q,n∈N∗\dfrac{a_{n + 1}}{a_n} = q, \quad n \in \N^\astan​an+1​​=q,n∈N∗，其中 qqq 是一个与 nnn 无关的常数．
  • 证明 an+1=qan,n∈N∗a_{n + 1} = qa_n, \quad n \in \N^\astan+1​=qan​,n∈N∗，其中 qqq 是一个与 nnn 无关的常数；且必须再验证 a1≠0a_1 \ne 0a1​=0，q≠0q \ne 0q=0．

方法一是等比数列的定义，证明每一项与前一项的比为一定值．这种方法并不需要多余验证．

来看一道例题：

例题 4.2

数列 aaa 满足 a1=1a_1 = 1a1​=1，an+1=2an+1,n∈N∗a_{n + 1} = 2a_n + 1, \quad n \in \N^\astan+1​=2an​+1,n∈N∗，求证：{an+1}\{a_n + 1\}{an​+1} 是等比数列．

例题 4.2 解答（方法一）

试图证明 an+1+1an+1\dfrac{a_{n + 1} + 1}{a_n + 1}an​+1an+1​+1​ 是定值．

an+1=2an+1a_{n + 1} = 2a_n + 1an+1​=2an​+1 可以简单变形得 an+1+1=2(an+1)a_{n + 1} + 1 = 2(a_n + 1)an+1​+1=2(an​+1)．

看起来胜利在望，只要证明 an+1≠0a_n + 1 \ne 0an​+1=0，两边同时除以这个式子就解决了．但这里对 an+1≠0a_n + 1 \ne 0an​+1=0 的证明需要采取数学归纳法，我们有更好的方案……

例题 4.2 解答（方法二）

an+1=2an+1a_{n + 1} = 2a_n + 1an+1​=2an​+1 可以简单变形得 an+1+1=2(an+1)a_{n + 1} + 1 = 2(a_n + 1)an+1​+1=2(an​+1)．

由于首项 a1+1=2≠0a_1 + 1 = 2 \ne 0a1​+1=2=0，公比 2≠02 \ne 02=0，因此数列为等比数列．

可以看到，方法二往往免去了证明待证数列任意一项不等于 000 可能造成的麻烦，而只需用验证首项和公比替代，这往往简单于前者．那么方法二的道理是什么呢？

其实还是数学归纳法．考虑 an+1=qana_{n + 1} = qa_nan+1​=qan​，a1≠0a_1 \ne 0a1​=0，q≠0q \ne 0q=0，我们有：

  • 对于任意 k∈N∗k \in \N^\astk∈N∗，如果 ak≠0a_k \ne 0ak​=0，则 ak+1=qak≠0a_{k + 1} = qa_k \ne 0ak+1​=qak​=0；
  • 而 a1≠0a_1 \ne 0a1​=0；
  • 因此对于所有 k∈N∗k \in \N^\astk∈N∗，ak≠0a_k \ne 0ak​=0；
  • 因此等式可以变形为 an+1an=q\dfrac{a_{n + 1}}{a_n} = qan​an+1​​=q．

也就是说采取方法二实际上是 帮你把上面这个小数学归纳法打包省略了，这在阅卷中是允许的．

当然这并不是说方法二一定简单于方法一，有时候待证数列任意一项不为 000 是显然的，比如已知 aaa 是等比数列证明 {2an}\{2a_n\}{2an​} 是等比数列，此时不用验证的方法一就更好一点．

值得注意的是，某些地区可能要求方法一也要验证．这在逻辑上是一步多余的操作，但应试教育就不要跟分数较劲了，推荐无论如何都写一下验证，最保险．
