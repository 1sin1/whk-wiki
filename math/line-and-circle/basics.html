方程表示曲线的定义​

平面解析几何是一门用含有 xxx，yyy 两个变元的单个方程表示平面直角坐标系上的曲线的一门学科．

对于平面直角坐标系上的一个曲线 Γ\GammaΓ，设曲线上的所有点集构成集合 AAA．对于含有 xxx，yyy 两个变元的单个方程 F(x,y)=0F(x, y) = 0F(x,y)=0，设该方程的解集构成集合 BBB．如果将 AAA 中的所有元素——点用平面直角坐标系上的坐标表示，那么 AAA 中的所有元素均为有序数对；如果将 BBB 中的元素——方程的解 {x=x0y=y0\begin{cases}x = x_0 \\ y = y_0\end{cases}{x=x0​y=y0​​ 用有序数对 (x0,y0)(x_0, y_0)(x0​,y0​) 表示，那么 BBB 中的所有元素也为有序数对．当且仅当 A=BA = BA=B 时，我们称方程 F(x,y)=0F(x, y) = 0F(x,y)=0 为 Γ\GammaΓ 的方程，Γ\GammaΓ 为方程 F(x,y)=0F(x, y) = 0F(x,y)=0 的图形．

上面采用 F(x,y)=0F(x, y) = 0F(x,y)=0 表示方程，是因为任何含有 xxx，yyy 两个变元的方程都可以变形为一个关于 x,yx, yx,y 的式等于 000 的形式．

重新梳理一下逻辑．设 AAA 是 Γ\GammaΓ 的点集的有序数对形式，BBB 是 F(x,y)=0F(x, y) = 0F(x,y)=0 的解集的有序数对形式，则 F(x,y)=0F(x, y) = 0F(x,y)=0 是 Γ\GammaΓ 的方程等价于 A=BA = BA=B．根据集合的知识，A=BA = BA=B 又等价于下面三个命题中的任一：

  • A⊆BA \subseteq BA⊆B 且 B⊆AB \subseteq AB⊆A，即对于 Γ\GammaΓ 上任意一点 P(x0,y0)P(x_0, y_0)P(x0​,y0​) 都有 F(x0,y0)=0F(x_0, y_0) = 0F(x0​,y0​)=0；且对于任何一组满足 F(x0,y0)=0F(x_0, y_0) = 0F(x0​,y0​)=0 的有序实数对 (x0,y0)(x_0, y_0)(x0​,y0​)，都有 P(x0,y0)P(x_0, y_0)P(x0​,y0​) 在 Γ\GammaΓ 上．
  • A⊆BA \subseteq BA⊆B 且 ∁UA∩B=∅\complement_UA \cap B = \varnothing∁U​A∩B=∅．即对于 Γ\GammaΓ 上任意一点 P(x0,y0)P(x_0, y_0)P(x0​,y0​) 都有 F(x0,y0)=0F(x_0, y_0) = 0F(x0​,y0​)=0；对于不在 Γ\GammaΓ 上的任意一点 P(x0,y0)P(x_0, y_0)P(x0​,y0​) 都有 F(x0,y0)≠0F(x_0, y_0) \ne 0F(x0​,y0​)=0．
  • B⊆AB \subseteq AB⊆A 且 ∁UB∩A=∅\complement_UB \cap A = \varnothing∁U​B∩A=∅．即对于任何满足 F(x0,y0)=0F(x_0, y_0) = 0F(x0​,y0​)=0 的实数 x0x_0x0​，y0y_0y0​，都有 P(x0,y0)P(x_0, y_0)P(x0​,y0​) 在 Γ\GammaΓ 上；对于任何满足 F(x0,y0)≠0F(x_0, y_0) \ne 0F(x0​,y0​)=0 的实数 x0x_0x0​，y0y_0y0​，都有 P(x0,y0)P(x_0, y_0)P(x0​,y0​) 不在 Γ\GammaΓ 上．

因此，F(x,y)=0F(x,y) = 0F(x,y)=0 是 Γ\GammaΓ 的方程和上面三个命题都是互相等价的．也即：

  • 如果我们知道 F(x,y)=0F(x, y) = 0F(x,y)=0 是 Γ\GammaΓ 的方程，我们可以得到上面三个命题，从而利用方程研究曲线和它上面的点．比如，假如我们知道 x3+y3=2x^3 + y^3 = 2x3+y3=2 是 Γ\GammaΓ 的方程，则：
    • 虽然我们不知道 Γ\GammaΓ 的形状如何，但我们可以知道 (1,1)(1, 1)(1,1) 一定在 Γ\GammaΓ 上，因为它满足 Γ\GammaΓ 的方程．
    • 我们又能知道 (2,2)(2, 2)(2,2) 一定不在 Γ\GammaΓ 上，因为它不满足 Γ\GammaΓ 的方程．
    • 而对于 Γ\GammaΓ 上的一个点 P(x0,y0)P(x_0, y_0)P(x0​,y0​)，我们又能知道一定有 x03+y03=2{x_0}^3 + {y_0}^3 = 2x0​3+y0​3=2．
  • 如果我们证明了上面三个命题的任一，我们就能证明 F(x,y)=0F(x, y) = 0F(x,y)=0 是 Γ\GammaΓ 的方程，从而证明方程是曲线的方程．
    • 比如证明 x2+y2=1x^2 + y^2 = 1x2+y2=1 的曲线是以原点为圆心，半径为 111 的圆：对于圆上的任意一点，它和圆点的距离一定是 111，即 x2+y2=1x^2 + y^2 = 1x2+y2=1；而对于圆内和圆外任意一点，它和圆点的距离一定不是 111，即 x2+y2≠1x^2 + y^2 \ne 1x2+y2=1，得证．这里我们用了上面的第二条证明，即证明曲线上任一点满足方程，再证明不在曲线上任一点都不满足方程．

两曲线和两方程的关系​

一个点是两个曲线的交点，等价于这个点同时满足两个曲线的方程．

据此，两个曲线的交点可以直接联立两个曲线的方程进行研究．

两曲线重合，等价于它们的方程等价．

上面两个结论证明不难，读者可以自己尝试．
