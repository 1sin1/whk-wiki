圆的方程

圆的方程内容​

圆的标准方程​

圆心为 A(x0,y0)A(x_0, y_0)A(x0​,y0​)，半径为 rrr 的圆 ⊙A\odot A⊙A 的方程为：(x−x0)2+(y−y0)2=r2(x - x_0)^2 + (y - y_0)^2 = r^2(x−x0​)2+(y−y0​)2=r2．

原因非常简单．任意一点 P(x,y)P(x, y)P(x,y) 在圆 ⊙A\odot A⊙A 上的充要条件为：P(x,y)P(x, y)P(x,y) 到 A(x0,y0)A(x_0, y_0)A(x0​,y0​) 的距离为 rrr，即 (x−x0)2+(y−y0)2=r\sqrt{(x - x_0)^2 + (y - y_0)^2} = r(x−x0​)2+(y−y0​)2​=r，整理得 (x−x0)2+(y−y0)2=r2(x - x_0)^2 + (y - y_0)^2 = r^2(x−x0​)2+(y−y0​)2=r2．

这个方程叫做 圆的标准方程，或 圆的标准式．

以后见到方程 (x−a)2+(y−b)2=c(x - a)^2 + (y - b)^2 = c(x−a)2+(y−b)2=c，其中 c>0c > 0c>0，就可以直接分析出它是一个圆，且圆心为 (a,b)(a, b)(a,b)，半径为 c\sqrt cc​（注意不是 ccc）．

圆的一般方程​

一般地，圆的标准方程 (x−a)2+(y−b)2=r2(x - a)^2 + (y - b)^2 = r^2(x−a)2+(y−b)2=r2 都可以变形为 x2+y2+Dx+Ey+F=0x^2 + y^2 + Dx + Ey + F = 0x2+y2+Dx+Ey+F=0 的形式．那么形如 x2+y2+Dx+Ey+F=0x^2 + y^2 + Dx + Ey + F = 0x2+y2+Dx+Ey+F=0 的方程都表示的是圆吗？并非如此．

考虑尝试将 x2+y2+Dx+Ey+F=0x^2 + y^2 + Dx + Ey + F = 0x2+y2+Dx+Ey+F=0 化为标准形式．根据配方，我们不难得到这个方程等价为 (x+D2)2+(y+E2)2=D2+E2−4F4\left(x + \dfrac D 2\right)^2 + \left(y + \dfrac E 2\right)^2 = \dfrac{D^2 + E^2 - 4F}{4}(x+2D​)2+(y+2E​)2=4D2+E2−4F​．于是，

  • 当 D2+E2−4F>0D^2 + E^2 - 4F > 0D2+E2−4F>0 时，该方程表示以 (−D2,−E2)\left(-\dfrac D 2, -\dfrac E 2\right)(−2D​,−2E​) 为圆心，D2+E2−4F4\sqrt{\dfrac{D^2 + E^2 - 4F}4}4D2+E2−4F​​ 为半径的圆．
  • 当 D2+E2−4F=0D^2 + E^2 - 4F = 0D2+E2−4F=0 时，该方程仅表示 (−D2,−E2)\left(-\dfrac D 2, -\dfrac E 2\right)(−2D​,−2E​) 这个点．
  • 当 D2+E2−4F<0D^2 + E^2 - 4F < 0D2+E2−4F<0，该方程无解，不表示任何图形．

综上，我们称 x2+y2+Dx+Ey+F=0x^2 + y^2 + Dx + Ey + F = 0x2+y2+Dx+Ey+F=0（其中 D2+E2−4F>0D^2 + E^2 - 4F > 0D2+E2−4F>0）叫做 圆的一般方程，或 圆的一般式．

圆的角度上分析任意二次方程​

对于任意二次方程 Ax2+By2+Cxy+D′x+E′y+F′=0Ax^2 + By^2 + Cxy + D'x + E'y + F' = 0Ax2+By2+Cxy+D′x+E′y+F′=0，满足什么条件时才是一个圆？如果是，它的半径和圆心如何分析？

首先，它必须得能变形成 x2+y2+Dx+Ey+F=0x^2 + y^2 + Dx + Ey + F = 0x2+y2+Dx+Ey+F=0，即圆的一般方程形式，否则，它必定无法进一步变形为圆的标准方程，进而无法表示圆．因此，我们有如下必要条件：

  • A=B≠0A = B \ne 0A=B=0．即两个平方项系数必须相等且不为 000．
  • C=0C = 0C=0，即方程不能有 xyxyxy 项．

有了这些条件后，方程两侧同时除以 AAA，即可变形为 x2+y2+Dx+Ey+F=0x^2 + y^2 + Dx + Ey + F = 0x2+y2+Dx+Ey+F=0．根据一般式方程的知识，再保证 D2+E2−4F>0D^2 + E^2 - 4F > 0D2+E2−4F>0，我们就能保证方程可以表示为一个圆．

总结：遇到一个二次方程，想在圆的角度分析它时，操作步骤如下：

  • 判定是否 A=B≠0A = B \ne 0A=B=0 且 C≠0C \ne 0C=0．
  • 如果是，进一步把方程变形为圆的一般方程的形式．
  • 再判断 D2+E2−4FD^2 + E^2 - 4FD2+E2−4F 与 000 的大小关系．
注意

**上面的第二，第三步不能调换．**必须将平方项系数变形为全为 111 的形式后，才能将 DDD，EEE，FFF 代入 D2+E2−4FD^2 + E^2 - 4FD2+E2−4F 判断正负．当方程平方项系数不全为 111 时，D′2+E′2−4F′{D'}^2 + {E'}^2 - 4F'D′2+E′2−4F′ 的正负不能决定方程是否表示圆．

上面已经将二次方程整理为圆的一般方程的形式．如果我们想进一步分析圆心坐标和半径，我们需要将它整理为圆的标准方程的形式．

如果读者是善记弱计算的选手，可以直接记结论．一般方程

x2+y2+Dx+Ey+F=0x^2 + y^2 + Dx + Ey + F = 0x2+y2+Dx+Ey+F=0

转换成标准方程的结果是

(x+D2)2+(y+E2)2=D2+E2−4F4\left(x + \dfrac D 2\right)^2 + \left(y + \dfrac E 2\right)^2 = \dfrac{D^2 + E^2 - 4F}{4}(x+2D​)2+(y+2E​)2=4D2+E2−4F​

从而得到圆心为 (−D2,−E2)\left(-\dfrac D 2, -\dfrac E 2\right)(−2D​,−2E​)，半径为 D2+E2−4F4\sqrt{\dfrac{D^2 + E^2 - 4F}4}4D2+E2−4F​​．这里的 D2+E2−4FD^2 + E^2 - 4FD2+E2−4F 刚刚为了确定它是圆应该已经算过一次了，那么这里直接代入即可．

例题 1.3

已知方程 a2x2+(a+2)y2+4x+8y+5a=0a^2x^2 + (a + 2)y^2 + 4x + 8y + 5a = 0a2x2+(a+2)y2+4x+8y+5a=0 表示圆，求圆的圆心坐标和半径．

例题 1.3 解答

首先我们要约束它是圆．要整理成一般方程，需要有 a2=a+2a^2 = a + 2a2=a+2，解得 a=2a = 2a=2 或 a=−1a = -1a=−1．

当 a=2a = 2a=2 时，方程整理为 x2+y2+x+2y+52=0x^2 + y^2 + x + 2y + \dfrac 5 2 = 0x2+y2+x+2y+25​=0．该方程 12+22−4×52<01^2 + 2^2 - 4 \times \dfrac 5 2 < 012+22−4×25​<0，不是圆．

当 a=−1a = -1a=−1 时，方程整理为 x2+y2+4x+8y−5=0x^2 + y^2 + 4x + 8y - 5 = 0x2+y2+4x+8y−5=0，该方程 42+82−4×(−5)=100>04^2 + 8^2 - 4 \times (-5) = 100 > 042+82−4×(−5)=100>0，是圆．直接写出配方结果为 (x+42)2+(y+82)2=1004\left(x + \dfrac 4 2\right)^2 + \left(y + \dfrac 8 2\right)^2 = \dfrac{100}4(x+24​)2+(y+28​)2=4100​，即 (x+2)2+(y+4)2=25(x + 2)^2 + (y + 4)^2 = 25(x+2)2+(y+4)2=25，直接得到圆心 (−2,−4)(-2, -4)(−2,−4) 半径为 555，即答案．

三点确定一圆（三角形外接圆）​

已知一个圆 ⊙P\odot P⊙P 经过点 A(x1,y1)A(x_1, y_1)A(x1​,y1​)，B(x2,y2)B(x_2, y_2)B(x2​,y2​)，C(x3,y3)C(x_3, y_3)C(x3​,y3​) 三点，求 PPP 的方程．

直接设 ⊙P\odot P⊙P 方程为 x2+y2+Dx+Ey+F=0x^2 + y^2 + Dx + Ey + F = 0x2+y2+Dx+Ey+F=0，然后将三个坐标分别代入即可．

这种做法是最简单的，因为三个坐标带入后，形成的是三个三元一次方程，可解且不难解．

对比之下，如果我们设 ⊙P\odot P⊙P 的方程为 (x−x0)2+(y−y0)2=r2(x - x_0)^2 + (y - y_0)^2 = r^2(x−x0​)2+(y−y0​)2=r2，代入三个坐标后形成的是三个二次方程，求解难度远大于上面的一次方程做法．

后面我们还会介绍另一种做法．

圆的方程题型​

圆的面积​

圆的面积是一个只与半径直接相关，与圆心无关的量．看到面积条件就可以刻画为半径条件．

例题 2.1

求方程 x2+y2+kx+2y+k2=0x^2 + y^2 + kx + 2y + k^2 = 0x2+y2+kx+2y+k2=0 所表示的圆面积最大时，该圆的圆心坐标．

例题 2.1

不难发现该圆半径为 k2+4−4k24=1−3k24\sqrt{\dfrac{k^2 + 4 - 4k^2}4} = \sqrt{1 - \dfrac{3k^2}4}4k2+4−4k2​​=1−43k2​​．该半径取最大值时，有 k=0k = 0k=0．

该圆圆心为 (−k2,−1)\left(-\dfrac k 2, -1\right)(−2k​,−1)．因此面积最大，即半径最大，即 k=0k = 0k=0，此时圆心为 (0,−1)(0, -1)(0,−1)．

圆关于直线对称​

圆关于 lll 对称，等价于圆上存在两点关于 lll 对称，等价于圆关于 lll 对称后的图形仍是其本身；它们都可以被刻画为圆心在 lll 上．与圆的面积恰好相反，这是一个与圆心有关而与半径无关的条件．

例题 2.2

若圆 x2+y2−2ax−2a2y+a4=0x^2 + y^2 - 2ax - 2a^2y + a^4 = 0x2+y2−2ax−2a2y+a4=0 上存在两个点关于 y=xy = xy=x 对称，求 aaa 的值．

例题 2.2 解答

该方程表示圆的条件为 (−2a)2+(−2a2)2−4a4>0  ⟺  a>0(-2a)^2 + (-2a^2)^2 - 4a^4 > 0 \iff a > 0(−2a)2+(−2a2)2−4a4>0⟺a>0．

条件等价于该圆的圆心在 y=xy = xy=x 上，即 (a,a2)\left(a, a^2\right)(a,a2) 在 y=xy = xy=x 上，因此 a2=aa^2 = aa2=a，联系 a>0a > 0a>0，可得 a=1a = 1a=1．

为什么例题 2.2 需要讨论该方程表示圆的条件，而例题 2.1 却忽略了？这是因为例题 2.1 已经讨论了半径，当计算出来的半径为正值时，暗含圆一定存在，因此可以忽略；而例题 2.2 中只有对圆心的讨论，因此不能忽略方程不表达圆的情形．

圆的直径式​

圆的直径一般做法​

已知圆的某条直径 ABABAB，可以刻画为圆心在 ABABAB 中点，且半径为 ∣AB∣2\dfrac{|AB|}22∣AB∣​．

圆的直径式做法​

直径端点分别为 A(x1,y1)A(x_1, y_1)A(x1​,y1​) 和 B(x2,y2)B(x_2, y_2)B(x2​,y2​) 的圆可以表示为 (x−x1)(x−x2)+(y−y1)(y−y2)=0(x - x_1)(x - x_2) + (y - y_1)(y - y_2) = 0(x−x1​)(x−x2​)+(y−y1​)(y−y2​)=0．

考虑平面直角坐标系中任意一点 P(x,y)P(x, y)P(x,y)．

  • 当它与 AAA 或与 BBB 重合时，显然 AP→⊥BP→\overrightarrow{AP} \perp \overrightarrow{BP}AP⊥BP（因为其中一个是零向量）．
  • 否则，它在圆上等价于 ∠APB=π2\angle APB = \dfrac \pi 2∠APB=2π​，即 AP→⊥BP→\overrightarrow{AP} \perp \overrightarrow{BP}AP⊥BP．

因此无论如何，PPP 在圆上都等价于 AP→⊥BP→\overrightarrow{AP} \perp \overrightarrow{BP}AP⊥BP，即 (x−x1,y−y1)⋅(x−x2,y−y2)=0(x - x_1, y - y_1) \cdot (x - x_2, y - y_2) = 0(x−x1​,y−y1​)⋅(x−x2​,y−y2​)=0．

据此我们根据直径端点的坐标直接列出圆的方程，这个方程叫圆的直径式方程．大题中不能直接使用．

公共弦所在直线方程​

在两圆 ⊙A\odot A⊙A，⊙B\odot B⊙B 恰交于 PPP，QQQ 两点时，称线段 PQPQPQ 为两圆公共弦．现在研究直线 PQPQPQ 的方程．

一个优于暴力求解，简单的算法如下：

  • 设 ⊙A\odot A⊙A 的一般式为 x2+y2+D1x+E1y+F1=0x^2 + y^2 + D_1x + E_1y + F_1 = 0x2+y2+D1​x+E1​y+F1​=0，⊙B\odot B⊙B 的一般式为 x2+y2+D2x+E2y+F2=0x^2 + y^2 + D_2x + E_2y + F_2 = 0x2+y2+D2​x+E2​y+F2​=0．
  • 则所求公共弦所在直线方程为上述两方程相减得到的方程，即 (D1−D2)x+(E1−E2)y+F1−F2=0(D_1 - D_2)x + (E_1 - E_2)y + F_1 - F_2 = 0(D1​−D2​)x+(E1​−E2​)y+F1​−F2​=0．

下面证明得到的方程确实是所求直线的方程．

两圆恰交于两点时，必定有 D1−D2≠0D_1 - D_2 \ne 0D1​−D2​=0 或 E1−E2≠0E_1 - E_2 \ne 0E1​−E2​=0．因此得到的方程表示一条直线．

考虑两圆的交点 PPP 和 QQQ．由于它们都各自满足两个圆的方程，因此也一定满足两个圆方程相减得到的方程．即 PPP 和 QQQ 在表示的直线上．证毕．

例题 2.4

求两圆 x2+y2−2x+10y−24=0x^2 + y^2 - 2x + 10y - 24 = 0x2+y2−2x+10y−24=0 与 x2+y2+2x+2y−8=0x^2 + y^2 + 2x + 2y - 8 = 0x2+y2+2x+2y−8=0 的公共弦所在直线方程．

例题 2.4 解答

两方程相减得 −4x+8y−16=0-4x + 8y - 16 = 0−4x+8y−16=0，整理得 x−2y+4=0x - 2y + 4 = 0x−2y+4=0．

两圆必须相交

当两圆的交点数量不为 222 时，上述算法也能求出一条直线方程，但这条直线方程显然与交点弦没什么关系——此时交点弦甚至是无定义的．

切点弦所在直线方程​

对于 ⊙C\odot C⊙C 和在该圆外的一点 PPP，从 PPP 出发可以作 ⊙C\odot C⊙C 的两条切线，从而形成两个切点（切线与圆的交点）AAA 和 BBB．ABABAB 连接得到的弦称作切点弦，现求解直线 ABABAB 的方程．

切点弦所在直线方程 公共弦法​

一个优于暴力求解，简单的算法如下：

连接 PCPCPC，作 PCPCPC 中点 MMM，不难发现 PPP，AAA，BBB，CCC 都在以 MMM 为圆心，MPMPMP 为半径的 ⊙M\odot M⊙M 上．从而，⊙M\odot M⊙M 和 ⊙C\odot C⊙C 的公共弦就是所求的切点弦．算出公共弦直线方程即可．

上面的 ⊙M\odot M⊙M 其实是以线段 PCPCPC 为直径作出的圆，因此可以利用直径式快速写出 ⊙M\odot M⊙M 的方程．

另外，以 PPP 为圆心，PAPAPA 为半径作圆，得到的 ⊙P\odot P⊙P 与 ⊙C\odot C⊙C 的公共弦也是所求切点弦．不过这里想得到 PAPAPA 的长度，还需要 PCPCPC 与半径 ACACAC 勾股一下．请读者依据自己喜好选择方法．

例题 2.5

过点 P(3,2)P(3, 2)P(3,2) 作 ⊙C\odot C⊙C：(x−1)2+y2=1(x - 1)^2 + y^2 = 1(x−1)2+y2=1 的两条切线，切点分别为 AAA，BBB，求直线 ABABAB 的方程．

例题 2.5 解答

由题可知 C(1,0)C(1, 0)C(1,0)，以 PCPCPC 为直径作圆 (x−3)(x−1)+y(y−2)=0(x - 3)(x - 1) + y(y - 2) = 0(x−3)(x−1)+y(y−2)=0，整理得 x2+y2−4x−2y+3=0x^2 + y^2 - 4x - 2y + 3 = 0x2+y2−4x−2y+3=0．⊙C\odot C⊙C 方程整理得 x2+y2−2x=0x^2 + y^2 - 2x = 0x2+y2−2x=0，两方程相减得公共弦所在直线方程 2x+2y−3=02x + 2y - 3 = 02x+2y−3=0，即所求切点弦所在直线方程．

切点弦所在直线方程 结论​

直线 ABABAB 的方程为 (x−xC)(xP−xC)+(y−yC)(yP−yC)=r2(x - x_C)(x_P - x_C) + (y - y_C)(y_P - y_C) = r^2(x−xC​)(xP​−xC​)+(y−yC​)(yP​−yC​)=r2．

隐距离​

类似一次分式可以转换成斜率，形如

Ax2+Ay2+D′x+E′y+F′Ax^2 + Ay^2 + D'x + E'y + F'Ax2+Ay2+D′x+E′y+F′

的式子可以转换成距离．

考虑将其变形为

A(x2+y2+Dx+Ey+F)A(x^2 + y^2 + Dx + Ey + F)A(x2+y2+Dx+Ey+F)

则括号内的部分可以变形为

(x+D2)2+(y+E2)2−D2+E2−4F4\left(x + \dfrac D 2\right)^2 + \left(y + \dfrac E 2\right)^2 - \dfrac{D^2 + E^2 - 4F}4(x+2D​)2+(y+2E​)2−4D2+E2−4F​

即 (x,y)(x, y)(x,y) 到 (−D2,−E2)\left(-\dfrac D2, -\dfrac E2\right)(−2D​,−2E​) 的距离的平方偏移一个常数．原式就是这个结果再扩大 AAA 倍的结果．

这样，我们就将形如 Ax2+Ay2+D′x+E′y+F′Ax^2 + Ay^2 + D'x + E'y + F'Ax2+Ay2+D′x+E′y+F′ 的式子转化为了距离．
